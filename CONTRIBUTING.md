# Contributing & Contact

Thank you for your interest in **Souq**. 👋

This repository is a **public portfolio and case study** (a map-first marketplace platform for the Gulf), not an open-source software
project. The application source code, infrastructure, and proprietary logic are **private**, so there
is no codebase here to open pull requests against.

That said, engagement is very welcome.

## Ways to engage

- **💡 Product/research feedback & ideas** — open an [Issue](../../issues) describing the idea or the
  problem it addresses. Roadmap suggestions are tracked here too.
- **🐞 Documentation issues** — spotted a typo, a broken link, or an unclear explanation in this
  showcase? Open an issue and it'll be fixed.
- **🤝 Partnerships & collaboration** — collaborators, investors, and potential partners are
  encouraged to reach out (see below).
- **🧭 Walkthrough requests** — a live demo or a deeper technical discussion can be arranged privately.

## What to expect

- This repo's content (docs, diagrams, screenshots) may receive documentation PRs; the **product**
  itself is built privately.
- Please **do not** request the private source code, credentials, or schema via issues — those are
  not shared publicly. Source access can be discussed for employment or collaboration under
  appropriate confidentiality terms.

## Code of conduct

Be respectful, constructive, and professional. Harassment or abuse of any kind is not tolerated in
this project's spaces.
